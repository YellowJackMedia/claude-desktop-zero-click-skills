# Claude Desktop Native Zero-Click Skills Pack

A Claude Desktop native skill pack for zero-click content strategy, SEO rewriting, and decision gating before writing.

## What this repo is

This repository packages reusable Claude Desktop skills for:

- choosing the right content format for a topic
- rewriting existing content for AI search and zero-click discovery
- forcing a quick decision gate before the model writes anything

It is designed to be dropped straight into Claude Desktop's skills folder on Windows or macOS.

## Why this matters

Search is changing fast. The pages that win now are not just blog posts:

- owned audience pages
- transaction pages
- original research
- community content
- creator video/podcast assets
- in-depth reviews and tests
- trust pages
- tools and calculators
- directories and databases

If you want Claude Desktop to help strategically, it needs a native skill layer that knows when to write, when to rewrite, and when to ask questions first.

## Included skills

- `zero-click-content-strategy` — picks the best content type for the job
- `zero-click-content-rewriter` — upgrades existing content for SEO, AI search, and trust
- `claude-desktop-decision-gate` — decides whether the task needs questions, a draft, a rewrite, or an audit

## Install in Claude Desktop

### Windows

1. Close Claude Desktop.
2. Copy the `skills/` folders into your Claude skills directory.
3. Restart Claude Desktop.

### Example path

`C:\Users\<you>\.claude\skills\`

### macOS

`~/Library/Application Support/Claude/skills/`

## Recommended workflow

1. Ask Claude Desktop what the content goal is.
2. Run the decision gate skill.
3. Use the strategy skill to pick the right asset type.
4. Use the rewriter skill if the content already exists.
