# Claude Desktop Zero-Click Skills Pack

This pack routes content work into the right native Claude Desktop skill:

- **zero-click-content-strategy** → pick the best asset type
- **zero-click-content-rewriter** → rewrite existing content
- **claude-desktop-decision-gate** → decide what to do before writing

## Use this pack when

- the user wants Claude Desktop-native content workflows
- a topic needs the right format, not just a blog post
- an existing page needs AI-search optimization
- the request is vague and needs a quick workflow decision

## Default behavior

1. Classify the request.
2. Ask only the minimum clarifying questions.
3. Choose the best skill.
4. Return a concrete next step.

## Success condition

The assistant chooses the right content path fast, without defaulting to generic blog writing.
