# Zero-Click Content Rewriter

Use this skill when the content already exists and needs a stronger shape for modern search.

## Primary goals

- improve answerability
- increase semantic clarity
- add trust signals
- tighten structure
- preserve voice

## Rewrite checklist

Before rewriting, check:
- Does the page answer the core question fast?
- Is the topic obvious in the title and intro?
- Are entities named clearly?
- Is the content easy for AI systems to extract?
- Does the page earn trust with proof, examples, or citations?

## Rewrite moves

- lead with the answer
- replace vague language with specific entities
- add useful subheads
- remove filler
- add comparison tables only when they help clarity
- add citations, examples, and internal links where appropriate
- keep the original intent intact unless the user asks for a pivot

## What to preserve

- the author's voice
- the intended audience
- the business goal
- any factual accuracy already present

## What to improve

- structure
- clarity
- topical coverage
- conversion cues
- extractable summaries
- trust signals

## Output format

Return:
1. short diagnosis
2. rewritten outline or draft
3. specific improvements made
4. one follow-up question if needed

## Guardrails

- Don't rewrite into generic SEO fluff.
- Don't add claims without evidence.
- Don't change the article's purpose unless asked.
- Don't hide the answer behind long setup.

## Success condition

The page becomes easier for humans and systems to understand at a glance.
